# Web Programlama Proje Ödevi
Bu proje, ASP.NET Core MVC kullanılarak geliştirilen bir Spor Salonu Yönetim ve Randevu Sistemidir. Sistem, spor salonu işletmelerinin antrenör yönetimi, üye randevuları, hizmet takibi ve yapay zeka destekli kişiselleştirilmiş egzersiz önerileri gibi ihtiyaçlarını karşılamak üzere tasarlanmıştır.
